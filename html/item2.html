What if you can get a song playlist that matches your mood perfectly and utilize the wonders of facial-recognition technology?

For one of our hackathons at Coding House, I teamed up with two great friends Brett and Mike to make a fun app that would utilize two powerful APIs: Spotify and Microsoft Oxford Face API. We were curious about how we could respond to a user once we receive their emotion data, and we thought music was the perfect answer.

The work was divided up with Mike handling the Microsoft API and some Angular on the front-end and Brett writing API calls to Spotify and building out the backend. I worked on the front-end and wrote in some Angular as well. The parts where it got tricky were with the time-limits for Spotify API and browser restrictions for accessing a user's video cameras.

Since my teammates were so amazing, we solved the problems in no time and moved on to spend a lot of love on the design and UI. It was a fun project and really got me interested in playing with more amazing APIs. (Source Code)

AngularJS

Angular handled the APIs JSON response so that the view was updated, and was used with jQuery to make HTTP requests to the Spotify and Microsoft API.

Spotify API

Returns all you need in terms of music data like artists, albums, and tracks, with the only downside being that your auth key times out within an hour.

Microsoft Oxford Face API

Takes a facial expression as an input, and returns emotions such as anger, contempt, disgust, fear, happiness, neutral, sadness, and surprise.
