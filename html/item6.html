This is an in-progress app that allows beer-lovers to learn everything they need to know about the brews they love and enjoy.

The BreweryDB API is REST based, allowing developers to grab all kinds of data about beer with a straightforward, easy to understand API. I liked this particular API because the docs are definitely straightforward

This is a simple (in-progress) app that lets people see a random beer (http://www.brewerydb.com/developers/docs-endpoint/beer_random). They can then input if they have sampled/not sampled that beer.
