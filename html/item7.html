Very quick app to help kids learn to add

Using Vanilla JavaScript, I built a math game with randomly generated math problems. Kids can enter their answer by clicking number buttons that appear below the equation. The numbers appear in an answer box following the = sign in the equation.

Vanilla JavaScript is a name to refer to using plain JavaScript without any additional libraries like jQuery. It was very important for me to do this in JavaScript in order to internalize the fact that I can do a lot of things without the need for additional JavaScript libraries. One awesome website I used for reference quite often was You Might Not Need jQuery
